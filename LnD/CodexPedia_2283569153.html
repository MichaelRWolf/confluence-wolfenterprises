Google Apps Script Program

This program is written in Typescript and saved in a file called main.ts.
It will prompt the user for a date, amount, and account, and use the class Transaction definition in the file Transaction.ts.

Installation

  1. Create a blank Google Apps Script project.

  2. Go to the project's Resources menu and select Add a new file.

  3. Enter Transaction.ts as the new file name and paste the following code:

class Transaction {
  date: Date;
  amount: number;
  account: string;

  constructor(date: Date, amount: number, account: string) {
    this.date = date;
    this.amount = amount;
    this.account = account;
  }
}

  1. Go to the project's Resources menu and select Add a new file.

  2. Enter main.ts as the new file name and paste the following code:

// Prompt the user for a date, amount, and account
const dateString = prompt("Enter a date:");
const amountString = prompt("Enter an amount:");
const accountString = prompt("Enter an account:");

// Create a Transaction object
const tr = new Transaction(
  new Date(dateString),
  Number(amountString),
  accountString
);

// Do something with the Transaction object
console.log(`Transaction details: ${tr.date}, ${tr.amount}, ${tr.account}`);

  1. Save the project and run the main.ts file.

  2. Follow the prompts to enter the date, amount, and account for the new transaction.
