Networking Event

Pre-event

At-event

Goal - Maximum amount of 10-minute conversations (i.e. 6 conversations per hour)

For every person:

  1. Show genuine interest in helping them.

  2. Gather definition of Treasure for them.

  3. Close

    1. Tell: I enjoyed our conversation

    2. Offer: Follow-up opportunity

    3. Set expectations:

      1. Email

      2. Offer time/date for 1-on-1

      3. Follow-up w/ <specific items>

Post-event

Within 12 hours:

  • mind dump to note in CRM

    • Title: Notes from <event>

  • Break big note into separate notes

    • New person - <name>

    • Conversation w/ <name>

  • email, as promised

Daylite

  • Start from Appointment

  • “New Note” button

    • Title → “Notes for <event>”

    • Notes for all interactions. May be cut/paste’d later into separate Person records.

    • Include list of people

    • Include link to event

    • Include (searchable) event description

New Person

Daylite

  • Start from relevant <context> (Appointment, Opportunity, or Project)

  • “New Note” button

  • Set Title → “New Person - <name> [@ <event>]

  • Add Links (on note)

    • Given: relevant context

    • “+ Add Link” → <name> → “New Person”

    • Keywords → “New Person”

    • Set Content (of note)

    • Memory joggers (conversations, connections, stories…)

    • Definition of Treasure

    • List of follow-up items (redundant to tasks, but helpful)

  • Add Tasks (to note)

    • Add Link to <context>

  • Save

  • Return to Note->Person

    • “Write letter” → “New Person” template → to email
