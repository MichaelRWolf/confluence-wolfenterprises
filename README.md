# Confluence Export - wolfenterprises

This repository contains the exported Confluence spaces from the `wolfenterprises` account.

## Contents

This export includes multiple Confluence spaces with their associated pages, attachments, images, and styles.

## Structure

Each directory represents a Confluence space, containing:
- HTML files for pages
- `attachments/` directories with uploaded files
- `images/` directories with embedded images
- `styles/` directories with CSS files
- `thumbnails/` directories for attachment previews
