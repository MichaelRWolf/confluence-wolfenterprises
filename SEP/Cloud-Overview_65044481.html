Research Issues

What is...?

  • ...Spring Boot?  Spring Framework (for Java apps) + Bootstrap (the scaffolding to get it running)
  • ...Maven? System to make it easy to build Java projects (and provide standard ways of doing things across projects)
  • ... "Cloud Foundry Container Runtime"?
  • ... "Kickstart Moneta Boot Starter"?
  • ... Moneta Boot?  JPMC 

Aha!

  • Moneta Boot == JPMC extensions to Spring Boot + Spring Cloud
  • Spring Boot == Spring (Java Framework) + Boot (Easy bootstrapping without creating extra framework)
    • Spring Boot makes it easy to create stand-alone, production-grade Spring based Applications that you can "just run".
    • We take an opinionated view of the Spring platform and third-party libraries so you can get started with minimum fuss. Most Spring Boot applications need very little Spring configuration
  • Maven - System to make it easy to build Java projects (and provide standard ways of doing things across projects/industries/companies)

Key Points

  • Everything is code - versioned, no "tribal knowledge", shared, low "bus factor"

    • Functional.  Algorithmic.  Iterative.  Execution.
  • Tools
    • Maven
